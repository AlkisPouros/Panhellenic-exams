# ΠΧ6. Εισαγωγή μαθημάτων

**Πρωτεύων Actor**: Διαχειριστής συστήματος
**Ενδιαφερόμενοι**
**Βαθμολογητής**: Θέλει να υπάρχουν διαθέσιμα τα μαθήματα για να μπορεί να καταχωρήσει τις βαθμολογίες, έχοντας πρώτα κάνει login.

## Βασική ροή
1. Ο διαχειριστής εισάγει τα μαθήματα των πανελλαδικών εξετάσεων σε μία λίστα 
   η οποία θα διατηρεί αργότερα με την καταχώρηση των βαθμολογιών τον βαθμό για κάθε υποψήφιο που έδωσε το αντίστοιχο μάθημα, συμπεριλαμβάνοντας τα στοιχεία του.
